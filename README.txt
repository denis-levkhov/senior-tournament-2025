Турнир алгоритмов.
Встреча (матч) двух алгоритмов заключается в разыгрывании T раундов игры типа Дилемма Заключенного.
1. Запуск и тестирование алгоритмов можно проводить через index.html
2. Отлаживать логику поведения алгоритмов можно с помощью команды alert('log')
3. Ошибки при выполнении скрипта можно отлавливать, например, через консоль в отладчике Firefox/Chrome (правая кнопка мыши -> inspect, вкладка console)
