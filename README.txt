1. Шаблон-скрипт для вашей команды находится в файле app/models/players/ladasport.js
2. Запуск и тестирование алгоритмов можно проводить через index.html
3. Отлаживать логику поведения алгоритмов можно с помощью команды alert('log')
4. Ошибки при выполнении скрипта можно отлавливать, например, через консоль в отладчике Firefox/Chrome (правая кнопка мыши -> inspect, вкладка console)
5. Все дополнительные вопросы представители бригад могут адресовать в ваццап/телеграм.
